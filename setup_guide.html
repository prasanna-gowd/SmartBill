0 Prerequisites

Make sure you have these installed before starting.

RequirementVersionCheck Command
Python 3.8 or higher python --version
pip Latest pip --version
Tkinter Built-in with Python python -c "import tkinter"
💡
Tkinter comes pre-installed with standard Python on Windows. If you installed Python from python.org, you're all set.
1 Verify Python Installation

Open a terminal (Command Prompt / PowerShell / VS Code Terminal) and run:

PS> python --version # Expected output: Python 3.x.x PS> pip --version # Expected output: pip 2x.x from ...
⚠️
If python is not recognized, make sure Python is added to your system PATH. Re-run the Python installer and check "Add Python to PATH".
2 Navigate to the Project Folder

Change directory to where SmartBill is located.

PS> cd C:\Users\prasa\Desktop\SmartBill # Verify you're in the right folder: PS> dir # You should see: main.py, modules/, database/, etc.
3 Install Dependencies

Install the required Python package for analytics charts.

# Required — enables analytics charts PS> pip install matplotlib # Optional — enables date picker in legacy analysis scripts PS> pip install tkcalendar
📦 Dependency Summary

✅
The app works even without matplotlib — analytics will show text-based stats instead of charts.
4 Run the Application

Launch SmartBill Pro with a single command.

PS> python main.py
🚀
The app will automatically create the database, seed 44 menu items, set up default user accounts, and open the login window.

📊 Dashboard (Landing Page)

After login, you'll see today's revenue, order count, average order value, and menu item count. Use the Quick Actions panel on the right to jump to any section.

B

🛒 Billing / POS — Place an Order

Click "Billing / POS" in the sidebar → Select a table → Browse categories or use the search bar → Click "+ Add" on items → Adjust quantities with +/− buttons → Set discount % → Choose payment method (Cash / Card / UPI) → Click "✓ Place Order".

C

🧾 Receipt

After placing an order, a formatted receipt popup appears. Click "💾 Save Receipt" to save it as a text file in the reports/ folder.

D

📋 Order History

View all past orders with date range and table filters. Double-click any row to see full order details. Click "📤 Export CSV" to download order data.

E

📝 Menu Manager

Add new items, edit prices, or soft-delete items. Filter by category or search by name. Select an item in the table and click "✏️ Edit" or "🗑️ Delete".

F

📈 Analytics

View four interactive charts: Top Selling Items, Monthly Revenue, Category Breakdown (pie chart), and Hourly Order Distribution. Switch between tabs.

G

⚙️ Settings

Configure restaurant name, tax rate, number of tables, and currency symbol. Changes are saved to the database and persist across restarts.

8 Troubleshooting

Common issues and how to fix them.

❌ "python is not recognized"

Python is not in your system PATH. Reinstall Python and check "Add Python to PATH" during installation, or use py instead of python.

PS> py main.py
❌ "ModuleNotFoundError: No module named 'matplotlib'"

matplotlib is not installed. Install it:

PS> pip install matplotlib
❌ "No module named 'tkinter'"

Tkinter is missing (rare on Windows). Reinstall Python from python.org and make sure "tcl/tk and IDLE" is checked during installation.

❌ Database errors / corrupted data

Delete the database and let the app recreate it:

PS> Remove-Item database\smartbill.db PS> python main.py # A fresh database will be created automatically
❌ App window is too small / cut off

The app requires a minimum resolution of 1200×700. Maximize the window or reduce your display scaling in Windows Settings → Display → Scale.
